# UI

## 概要

Knowledge Hub は、情報へ素早くアクセスし、ストレスなく記録・閲覧できるユーザーインターフェースを目指す。

デスクトップ・タブレット・スマートフォンで快適に利用できるレスポンシブデザインを採用する。

---

# 画面一覧

| 画面     | 説明             |
| -------- | ---------------- |
| Home     | ホーム画面       |
| Notes    | ノート一覧       |
| Note     | ノート閲覧・編集 |
| Search   | 検索             |
| Tags     | タグ一覧         |
| Media    | メディア管理     |
| Settings | 設定             |

---

# Home

## 目的

Knowledge Hub の入口となる画面。

## 表示内容

- 最近更新したノート
- ピン留めしたノート
- 最近追加したメディア
- タグ一覧
- クイック検索

---

# Notes

## 目的

すべてのノートを一覧表示する。

## 機能

- ノート一覧
- 並び替え
- 作成日時表示
- 更新日時表示
- ノート作成
- ノート削除

---

# Note

## 目的

ノートの閲覧・編集を行う。

## レイアウト

```text
+---------------------------------------------------------+
| タイトル                                                |
+---------------------------------------------------------+
| 編集 | 閲覧 | 分割表示                                  |
+---------------------------------------------------------+
|                                                   |     |
| Markdown Editor                                  | 情報 |
|                                                   |-----|
|                                                   |タグ |
|                                                   |-----|
|                                                   |添付 |
|                                                   |-----|
|                                                   |リンク|
|                                                   |-----|
|                                                   |バック|
+---------------------------------------------------------+
```

## 機能

- Markdown編集
- Markdownプレビュー
- 分割表示
- タグ編集
- 添付ファイル管理
- ノートリンク
- バックリンク表示

---

# Search

## 目的

必要な情報を素早く見つける。

## 機能

- 全文検索
- タグ検索
- ファイル検索
- 検索結果一覧

---

# Tags

## 目的

タグからノートを探す。

## 機能

- タグ一覧
- タグ検索
- タグごとのノート一覧

---

# Media

## 機能

- ダウンロード
- 削除

```text
+---------------------------------------------------+
```

## サイドバー

表示内容

- ノート一覧
- 最近開いたノート
- タグ
- お気に入り

---

## 共通操作

- Ctrl + K：検索
- Ctrl + S：保存
- Ctrl + N：新規ノート

---

# レスポンシブ対応

## Desktop

- サイドバー常時表示
- エディタとプレビューの分割表示

## Tablet

- サイドバー折りたたみ
- 分割表示対応

## Mobile

- サイドバーをドロワー表示
- 編集画面を全画面表示
- タップ操作を優先したUI

---

# デザイン方針

- シンプルで直感的
- 情報を見つけやすいレイアウト
- キーボード操作を重視
- ダークモード対応
- アクセシビリティを考慮
- 一貫性のあるデザイン

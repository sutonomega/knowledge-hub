# Roadmap

## 開発方針

Knowledge Hub は、小さな機能を積み重ねながら成長させるプロジェクトです。

各バージョンで「実際に使える状態」を目標とし、完成度よりも継続的な改善を重視します。

---

# v0.1.0 - Minimum Viable Product (MVP)

**目標**

Markdownノートを作成・閲覧・管理できる。

### 機能

- [ ] ノート一覧
- [ ] ノート作成
- [ ] ノート編集
- [ ] ノート削除
- [ ] Markdown閲覧モード
- [ ] タグ機能
- [ ] 全文検索

---

# v0.2.0 - Knowledge Management

**目標**

ノート同士をつなげて知識を整理できる。

### 機能

- [ ] `[[ノートリンク]]`
- [ ] バックリンク
- [ ] 最近更新したノート
- [ ] お気に入り
- [ ] MOC（Map of Contents）

---

# v0.3.0 - Media Hub

- [ ] PDF表示
- [ ] 添付ファイル一覧

# 開発ポリシー

- 小さく作る
- 実際に使いながら改善する
- データ互換性を重視する
- シンプルな設計を維持する
- ユーザー（自分）の使いやすさを最優先にする
